# 🚀 Go (Golang) Mini Projects

This repository contains beginner-friendly projects built while learning Go (Golang).  
These projects helped me understand the basics of programming, logic building, and CLI applications.

---

## 🎮 Projects Included

### 🏧 ATM Simulation
A simple banking system that allows users to:
- Withdraw money
- Deposit money
- Check balance

### 🎯 Number Guessing Game
A game where the user tries to guess a randomly generated number.

### 🪓 Hangman Game
Classic word guessing game with lives system and ASCII-based interface.

### 🎮 Rock Paper Scissors
Play against the computer with random AI choices.
